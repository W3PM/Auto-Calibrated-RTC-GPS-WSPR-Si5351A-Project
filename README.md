# Auto-Calibrated RTC GPS WSPR Si5351A Project

This is a full featured WSPR signal source using the very popular Si5351A clock generator board. Frequency and time accuracy are maintained by a highly accurate temperature compensated DS3231 real time clock (RTC) board. Extensive menu options allow the user to change all necessary parameters in addition to selecting a built in SWR bargraph indicator for antenna adjustments.
